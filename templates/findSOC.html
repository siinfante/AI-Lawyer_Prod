How to use the tool

Steps to select the right SOC Code for you

  • Enter your ‘Job Description’
  • Click on the ‘Find SOC’ button to get your appropriate ’SOC’ code.
  • You can select the ‘SOC’ code from the options displayed below and click on the ’Submit’ button to store the ‘Job Description’ you have entered.
  • If you do not see the matching ’SOC’ code you can select ‘None of the above’ button.
  • You will get a text box where you can enter your correct ‘SOC’ code and submit the ‘Job Description’ which will be stored in the data base with the code you have entered.
*Note: Once the SOC Code submitted, you will not be able to alter the data, it will be submitted and stored to database.
About the tool

A Skilled Worker (SW) Visa is one of the most applied for visa categories for immigration in the UK. While preparing an SW visa application, an immigration lawyer has to perform a number of checks starting from the case risk assessment to the required legal activities, customer handling, application preparation and finally the submission. The visa application’s preparation takes time and manual effort, as for every job profile, a different eligibility check has to be made according to the UK Immigration System. Different occupations/ job profiles are categorized under the federal statistical standard called the Standard Occupation Classification (SOC). Each job category is assigned an SOC code which defines the benchmark eligibility criteria for an applicant to apply for a SW visa.

For more information please refer to :

https://www.ons.gov.uk/methodology/classificationsandstandards/standardoccupationalclassificationsoc

To explore the list of UK SOC codes, please refer to :

https://www.gov.uk/government/publications/skilled-worker-visa-eligible-occupations/skilled-worker-visa-eligible-occupations-and-codes

For every SW visa application, one of the main tasks that a caseworker performs is going through the Job Description/Duties/Roles in the applicant’s job offer and then match it to the correct SOC code manually. We have come up with a solution that automates this task for them with the help of Machine Learning.

Find the SOC code for an applicant on the basis of its Job Profile with a click of a button.

There are a number of SOC codes. We have started with classifying 26 SOC Codes and as we move forward with more data collection, we intend to accommodate all the codes in our tool.

All the information submitted is completely anonymised and purely used for research purpose.
